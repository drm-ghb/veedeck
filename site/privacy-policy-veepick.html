Privacy Policy — Veepick

Last updated: July 1, 2026

1. Overview

Veepick is a browser extension that allows interior designers and their teams to add products from any online shop directly to their shopping lists in veedeck.com. This policy explains what data the extension accesses, how it is used, and what is never collected.

2. Data the extension reads

When you activate the extension on a product page, it reads the following information from that page:

  • Product name, price, image URL
  • Brand / manufacturer name
  • Product color and dimensions (if available in page metadata)
  • Short product description
  • Page URL (used as the product source link)
  • Website hostname (used as the supplier name)

This data is extracted from publicly available page metadata (Open Graph tags, JSON-LD structured data) and is only processed locally in your browser and sent directly to your own veedeck account.

3. Data stored locally

The extension stores the following data in your browser's local storage (chrome.storage.local):

  • API key — your personal veedeck API key, used to authenticate requests to your account
  • App URL — the base URL of your veedeck instance
  • Selected image URL — temporarily stored when you pick a product image, cleared after use

This data never leaves your browser except as part of authenticated API calls to your own veedeck account.

4. Data sent to veedeck

When you click "Add to list", the product data you have reviewed and optionally edited is sent via HTTPS to the veedeck API associated with your account. This data is stored in your veedeck account and is subject to veedeck's own Privacy Policy.

No data is sent to any third party. No analytics, crash reporting, or telemetry services are used by this extension.

5. Host permissions

The extension requests access to all websites (<all_urls>). This permission is required because interior designers need to add products from any online shop — there is no fixed list of supported stores. The extension only reads product metadata when you explicitly activate it on a page. It does not run in the background on websites, does not track your browsing history, and does not store or transmit any page content beyond the product fields listed in Section 2.

6. Data we do not collect

Veepick does not collect, store, or transmit:

  • Browsing history or visited URLs (other than the current product page URL when you add a product)
  • Personal identification data
  • Cookies or session data from visited websites
  • Any data from pages where the extension is not actively used

7. Children's privacy

Veepick is a professional tool intended for adults. It does not knowingly collect data from children under 13 years of age.

8. Changes to this policy

If this privacy policy changes, the updated version will be published at this URL with a new "Last updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.

9. Contact

If you have any questions about this privacy policy or how Veepick handles your data, contact us at: contact@veedeck.com

Polityka Prywatności — Veepick

Ostatnia aktualizacja: 1 lipca 2026

1. Wstęp

Veepick to rozszerzenie przeglądarki umożliwiające projektantom wnętrz i ich zespołom dodawanie produktów z dowolnego sklepu internetowego bezpośrednio do list zakupowych w veedeck.com. Niniejsza polityka wyjaśnia, do jakich danych rozszerzenie ma dostęp, w jaki sposób są one wykorzystywane oraz czego rozszerzenie nigdy nie zbiera.

2. Dane odczytywane przez rozszerzenie

Po aktywowaniu rozszerzenia na stronie produktu odczytuje ono następujące informacje z tej strony:

  • Nazwa produktu, cena, adres URL zdjęcia
  • Marka / producent
  • Kolor i wymiary produktu (jeśli dostępne w metadanych strony)
  • Krótki opis produktu
  • Adres URL strony (używany jako link do źródła produktu)
  • Nazwa domeny (używana jako nazwa dostawcy)

Dane są pozyskiwane z publicznie dostępnych metadanych strony (tagi Open Graph, dane strukturalne JSON-LD) i są przetwarzane wyłącznie lokalnie w przeglądarce użytkownika, a następnie przesyłane bezpośrednio na jego konto veedeck.

3. Dane przechowywane lokalnie

Rozszerzenie przechowuje następujące dane w lokalnym magazynie przeglądarki (chrome.storage.local):

  • Klucz API — osobisty klucz API veedeck użytkownika, służący do uwierzytelniania żądań
  • Adres aplikacji — bazowy adres URL instancji veedeck użytkownika
  • Adres URL wybranego zdjęcia — przechowywany tymczasowo podczas wybierania zdjęcia produktu, usuwany po użyciu

Dane te nie opuszczają przeglądarki użytkownika, z wyjątkiem uwierzytelnionych wywołań API do jego własnego konta veedeck.

4. Dane przesyłane do veedeck

Po kliknięciu „Dodaj do listy" dane produktu — przejrzane i opcjonalnie zedytowane przez użytkownika — są przesyłane przez HTTPS do API veedeck powiązanego z kontem użytkownika. Dane te są przechowywane na koncie veedeck i podlegają Polityce Prywatności veedeck.

Żadne dane nie są przesyłane do podmiotów trzecich. Rozszerzenie nie korzysta z żadnych narzędzi analitycznych, raportowania błędów ani telemetrii.

5. Uprawnienia do wszystkich stron

Rozszerzenie żąda dostępu do wszystkich stron internetowych (<all_urls>). Uprawnienie to jest niezbędne, ponieważ projektanci wnętrz potrzebują możliwości dodawania produktów z dowolnego sklepu online — nie istnieje stała lista obsługiwanych sklepów. Rozszerzenie odczytuje metadane produktu wyłącznie wtedy, gdy użytkownik aktywuje je na stronie. Nie działa w tle na odwiedzanych stronach, nie śledzi historii przeglądania i nie przechowuje ani nie przesyła żadnych treści stron poza polami produktu wymienionymi w sekcji 2.

6. Dane, których nie zbieramy

Veepick nie zbiera, nie przechowuje ani nie przesyła:

  • Historii przeglądania ani odwiedzanych adresów URL (z wyjątkiem adresu URL bieżącej strony produktu podczas dodawania produktu)
  • Danych osobowych użytkownika
  • Plików cookie ani danych sesji z odwiedzanych stron
  • Jakichkolwiek danych ze stron, na których rozszerzenie nie jest aktywnie używane

7. Ochrona dzieci

Veepick jest narzędziem profesjonalnym przeznaczonym dla dorosłych. Rozszerzenie nie zbiera świadomie danych od dzieci poniżej 13. roku życia.

8. Zmiany polityki prywatności

W przypadku zmiany niniejszej polityki zaktualizowana wersja zostanie opublikowana pod tym adresem URL z nową datą „Ostatniej aktualizacji". Dalsze korzystanie z rozszerzenia po wprowadzeniu zmian oznacza akceptację zaktualizowanej polityki.

9. Kontakt

W razie pytań dotyczących niniejszej polityki prywatności lub sposobu przetwarzania danych przez Veepick prosimy o kontakt: contact@veedeck.com
