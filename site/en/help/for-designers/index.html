For designers

A guide to all modules in the designer panel: ProjectFlow, shopping lists, clients, contractors, the veepick plugin and more.

space_dashboardMain dashboard

Your starting dashboard — at a glance you see what needs attention: number of clients, active projects and shopping lists, recent projects with render preview, tasks for today and overdue ones, unread comments/pins, requests to change status or restore a version, and unread chat messages.

Clicking any element takes you straight to the right place in the app.

push_pinProjectFlow - renders & visualisations

The heart of veedeck: upload renders, organise them into rooms and collect live client feedback. Structure: Project → Rooms (e.g. Living room, Kitchen) → (optional folders) → Renders.

How to add a new project:

  1. Open "ProjectFlow" in the sidebar.
  2. Click "New project".
  3. Enter a title and optionally assign a client from the database.
  4. Click "Create".

How to add a render to a room:

  1. Open a project.
  2. Click the room you want to add a file to (or create a new one with the "+" icon).
  3. Click "Add render" or drag the file onto the upload area.

Supported formats: JPEG, PNG, WebP, HEIC and PDF. Render statuses: Under review → Accepted / Rejected. Uploading a new file to an existing render creates the next version — previous ones are preserved in the history and can be compared or restored.

Comments and pins: the client (if you allow it) clicks directly on the image to leave a note at a specific spot. You see every comment with a status of New / In progress / Done and can reply in the thread.

How to share a project with a client:

  1. Go to the "Clients" module.
  2. Open the client profile → "Contacts" tab.
  3. Select a contact person (or add a new one) and assign them to the project.

What you can configure per project (Project settings): whether the client can comment and place pins, independently accept renders, change status without requesting approval, restore older versions, upload their own files, and which modules are visible to them.

local_mallShopping lists

Product breakdowns (furniture, lighting, fixtures, etc.) for client approval. Structure: List → Sections → Products.

How to add a product:

  1. Open a list and select (or create) a section.
  2. Click "+ Add product".
  3. Paste a shop link, fill in data manually, pick from the product library, or use the veepick plugin.

Faster product adding — veepick plugin: instead of copying data by hand, install → veepick in the Chrome Web Store. When you're on a product page in a shop, click the plugin icon, select the shopping list and confirm — the data (name, image, price, link) fills in automatically.

Product statuses are set by the client: no status → Accepted / Rejected. The list can be exported to PDF or CSV (list menu → three dots), with full product data and — if you share prices — totals.

groupClients

Your contact and investor database: add and edit clients, contact persons (each with their own login account), assign to projects, archive.

Email invitation: client profile → "Contacts" → select/add a person → "Send invitation". The client receives an email with a link to set their password (valid 24 h).

Manual account creation: client profile → "Contacts" → "Create account" and enter the credentials you share directly with the client.

engineeringContractors

How to add a contractor and assign them to a project:

  1. Sidebar → "Contractors" → "Add contractor".
  2. Enter a name or company name.
  3. Open the profile → "Assign project" and select a project from the list.
  4. In the assignment, add folders and upload files.

Folders have three types: Drawings, Visualisations, Other — you can also link a folder directly from ProjectFlow (syncs automatically) or hide it from the contractor.

You create the contractor's account in their profile ("Create account") — you share the login credentials directly with them; the app does not send passwords by email automatically.

check_boxTasks

A simple task list with subtasks, priorities (Low / Medium / High) and statuses (To do / In progress / Done). Tasks are private — the client cannot see them. You can assign a task to a project, client or team member.

assignmentSurveys

How to send a survey:

  1. Sidebar → "Surveys" → "New survey" (from scratch or from a template).
  2. Add questions (short/long answer, single/multiple choice, rating, yes/no, budget range).
  3. Assign to a client (optional).
  4. Set the status to "Active" — otherwise the survey stays as a "Draft" and the client cannot see it.

The survey appears automatically in the assigned client's panel — no link needs to be sent. You'll see responses in the "Responses" tab.

calendar_monthCalendar & Notepad

Calendar — events with date, time, description and a guest list (adding guests from the client database from the Studio plan). Notepad — notes with a text editor and attachments, private to you and your team.

chat_bubbleDiscussions

A built-in messenger for internal conversations, project conversations (with clients) and conversations with contractors — in real time, without switching to email or external messaging apps.

package_2Product library

A global product database for your workspace. Products are added here automatically when you add them to a shopping list (via link or manually) or save them with the veepick plugin — once saved, a product can be reused across multiple projects.

extensionveepick plugin

A browser extension that lets you add products from any online shop to shopping lists without copying data by hand. Available in all plans.

Installation:

  1. Install the veepick plugin from the Chrome Web Store (or Opera Add-ons).
  2. Open the plugin panel and log in with the access key from Settings → Plugin (veepick).
  3. Done — the plugin is connected to your account.

How to add a product via veepick:

  1. Go to the product page in a shop.
  2. Click the veepick icon in the browser toolbar.
  3. Select a shopping list.
  4. Confirm — the product data fills in automatically.

If the plugin doesn't fetch data from the page or shows an authorisation error — check in Settings whether the access key is up to date and try generating a new one.

settingsSettings & account

In Settings you manage: Profile (account details, photo), Branding (logo and welcome message visible in the client panel), Appearance (theme, light/dark/system mode, PL/EN language, module visibility), Users (team and permissions), Notifications (email summaries, modules and frequency), Plugin (access key), Plan & billing, and Account (password, account deletion).

Glossary

Glossary

Project
One design commission for one client — contains rooms, renders and shopping lists.
Room
A space within a project (e.g. living room, kitchen) that groups renders.
Render
A visualisation file (image or PDF) uploaded to a room, with a status and version history.
Pin
A comment attached to a specific location on a render.
Shopping list
A product breakdown prepared for the client, divided into sections.
Product library
An archive of all saved products, ready to reuse in future projects.
Workspace
Your working environment — all projects, clients and settings.
Trial period
30 days of free access after registration, no credit card required.
Client panel
A simplified project view for the client, without designer tools.
Contractor panel
A view for the contractor — only their folders and files.
Assignment
The link between a contractor and a specific project.
veepick
A browser extension that adds products from online shops directly to shopping lists.
